Tool Assisted Literature Surveys - A Requirements Review

Systematic literature reviews follow a certain structure with specific stages and tasks:

 

Stage I:

  • Planning a review
  • Defining a scope

Stage II:

  • Search
  • Select
  • Evaluate

Stage III:

  • Analysis
  • Synthesis

Stage IV:

  • Reporting

 

Our goal is to validate a set of requirements for tools that aid in successfully conducting a literature survey. If we know these requirements, we can

  1. find tools that adress specific tasks,
  2. find out where no given tool excells yet,
  3. compose a set of interoperable tools,

to

  1. increase a researcher's efficiency in conducting structured literature reviews,
  2. help developers to know which features are missing,
  3. move towards collaborative, open science knowledge distribution.

 

We want to invite you to review these requirements for the aforementioned 8 tasks.

There are 28 questions in this survey.
